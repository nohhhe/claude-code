1-1.클로드 코드 설치하기
=========================
### ◉ 클로드 코드란?
클로드 코드는 앤트로픽에서 만든 에이전트형 인공지능 도구로, 사용자의 터미널 환경에서 직접 작동한다는 특징이 있습니다. 사용자 터미널에 접근할 수 있다는 것은 각종 인증 정보를 얻을 수 있다는 것이며 이러한 특징 외에도 사용자 고유의 개발 환경과 도구를 함께 이용할 수도 있습니다.

따라서 웹에서 이용할 수 있는 ChapGPT와는 다르게 물어보고 대답하는 과정을 연결하여 코드 편집, 오류 수정, 명령 실행, 커밋 생성, 테스트 실행 등의 전체 개발 워크플로를 완전히 자동화할 수 있습니다.

## ⭕ 클로드 코드 설치
클로드 코드는 macOS와 윈도우에서 모두 사용 가능하지만 개인적으로는 macOS 사용을 추천합니다. 윈도우 같은 경우에는 WSL2를 사용해서 새로운 환경을 구성해야 하고 디렉터리에 접근하는 구조가 유닉스/리눅스와 다르기 때문입니다.

#### ✦ 클로드 요금제에서 참고할 만한 내용
- 영어 비해서 한국어가 토큰을 많이 사용한다.
  - LLM은 텍스트를 "토큰"이라는 단위로 쪼개서 처리하는데 자주 나오는 단어나 문자 조합을 하나의 토큰으로 학습
  - 한국어는 자음+모음 조합, 한자어, 외래어 등 다양한 문자 조합이 많아서 토큰 수가 더 많아짐
  - 예를 들어 "안녕하세요"는 영어로 "Hello"보다 더 많은 토큰으로 쪼개질 수 있음

### ◉ macOS에서 클로드 코드 설치
1. Node.js 설치
2. 클로드 코드 설치
> npm install -g @anthropic-ai/claude-code
> claude -v

## ⭕ 클로드 코드 실행
원하는 디렉터리에서 터미널을 열고 아래 명령어를 입력합니다.
> claude

클로드 코드를 처음 실행하면 터미널 테마를 설정할 수 있고 기본값은 다크 모드 입니다.

클로드 코드는 실수할 수 있고 보안적으로 의심되는 코드는 실행하지 말라라는 경고 메시지를 출력합니다.

#### ❓ 클로드 코드는 보안상 이슈가 있지 않을까?
- 개인 계정 (Free/Pro/Max)
  - 기본적으로 학습 사용 여부를 선택할 수 있고, 설정에서 언제든 변경 가능합니다.
  - 학습 허용 시 5년 보관, 비허용 시 30일 후 삭제
  - 학습에 사용되더라도 사용자 ID(이메일)와 분리되어 처리되고, 제3자에 판매되지 않습니다.
- 비즈니스/API 계정
  - Team, Enterprise, API 사용자는 절대 학습에 사용하지 않습니다.
  - 비즈니스 계정은 명시적으로 데이터 학습을 금지하며 더 강력한 기밀성 보장을 제공합니다.
- 대응 방안
  - 개인 계정 사용 중: 설정에서 "모델 개선에 데이터 사용" 끄기
    - https://claude.ai/settings/data-privacy-controls 에 접속하여 설정을 off할 수 있다.
  - 회사 기밀 코드: API 또는 Enterprise 계정 사용 고려
  - 민감한 작업: Incognito 모드 사용 (학습에 절대 사용 안 됨)
    - 대화 내역이 저장되지 않고 메모리에도 남지 않으며 절대 학습에 사용되지 않는 일회용 대화 모드
    - 해당 모드를 사용하고 컨텍스트 압축을 통해서 다음 세션에서 이전 대화 내용을 참조할 수 있다.
