1-3.인공지능으로 내 컴퓨터에만 있는 정보 분석하기
=========================
다른 인공지능 도구와 다르게 클로드 코드와 같은 인공지능 에이전트의 가장 큰 강점은 현재 시스템의 정보를 읽을 수 있다는 점이다.

### ◉ 내가 사용하는 데스크탑/랩탑의 정보를 알아내기
- 하드웨어 정보, 운영 체제 정보를 물어보면 배시 명령어를 통해서 확인 후 정리하여 알려준다.

### 로컬 파일들의 정보를 읽고 계산하기
- 현재 디렉터리의 파일 목록을 읽고 빈 파일 읽거나 특정 파일들을 읽고 분석하여 결과를 알려준다.
